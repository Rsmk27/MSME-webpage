Pin diagrams, connector maps, and key specifications for the STM32F446RE microcontroller and NUCLEO-F446RE development board.

💻 STM32F446RE Microcontroller

The STM32F446RE is a high-performance ARM Cortex-M4 microcontroller from STMicroelectronics, featuring DSP instructions and FPU, making it ideal for motor control and IoT applications.

▶ NUCLEO-F446RE Development Board

The NUCLEO-F446RE is an affordable development board from ST with an integrated ST-LINK/V2-1 debugger, Arduino Uno R3-compatible headers, and ST Morpho extension connectors.

Board Overview

STM32 NUCLEO-F446RE board — MB1136 revision. Key connectors and components are labeled.

NUCLEO-F411RE Pin Map — Arduino Headers

NUCLEO-F411RE Arduino-compatible header pinout. The F446RE shares the same connector layout.

Serial (UART) Connections

NUCLEO F411RE/F446RE — Serial1 Rx/Tx connection detail. The ST-LINK VCP uses USART2 by default (PA2 TX, PA3 RX on F446RE).

ℹ On the F446RE, the default ST-LINK VCP (Virtual COM Port) maps to USART2 (PA2 = TX, PA3 = RX). Open a serial terminal at 115200 baud to see printf output.
